Aquí está organizado solo por módulos del sistema administrativo real, con descripción verbal para programadores y sin hablar de código.

TEMA: Módulos del sistema administrativo real para negocio de estética

---

REQUERIMIENTO GENERAL DEL SISTEMA

El sistema administrativo debe permitir registrar diariamente la información del negocio y mantener organizada la operación interna para la dueña, trabajadores, encargados y contadora.

El sistema debe poder usarse desde iPad, tablet, celular, computadora o laptop, sin depender de Excel instalado.

El idioma principal debe ser inglés, con posibilidad de traducción.

MÓDULO 13: Productos descontables en servicios

Descripción:
Crear una función para descontar productos usados en cada servicio antes de calcular comisiones.

Funcionalidades:

- Registrar si un servicio usa producto descontable.
- Registrar tipo de producto usado.
- Registrar cantidad usada.
- Registrar valor unitario del producto.
- Calcular el descuento total del producto.
- Calcular la base real para comisión después del descuento.
- Calcular cuánto corresponde al empleado.
- Calcular cuánto queda para el negocio.
- Aplicar descuento de tinte.
- Aplicar descuento de descolorante por cuchara.
- Aplicar descuento de keratina, amino o alisado por onza.

Reglas mencionadas:

- Si un servicio de tinte cuesta 150 florines y se descuentan 15 florines de producto, quedan 135 florines para dividir.
- El restante se divide 50% para el empleado y 50% para el negocio.
- Si usan dos cucharas de descolorante a 15 florines cada una, se descuentan 30 florines.
- Si usan dos onzas de keratina a 40 florines cada una, se descuentan 80 florines.
